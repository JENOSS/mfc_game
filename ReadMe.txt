# mfc_game
리듬 액션 게임 - MFC를 이용한 리듬액션게임

# About
- FC를 이용한 리듬액션게임
- 원하는 wav파일을 등록 후 게임 생성, 박자에 맞춰 키보드 입력
- 키보드 입력 순간 오브젝트의 위치에 따라 점수와 이펙트에 차이를 둠
- 분할 윈도우로 게임이 진행되는 뷰와 점수, 랭킹, 체력 게이지가 출력되는 뷰로 나눔
- Double buffering을 구현하여 Flickering(깜빡거림) 현상을 해결
